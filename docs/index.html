RoboCup Tactical Analysis

Interactive analysis reports for a final year project based on the RoboCup 2D simulation environment. The reports compare baseline HELIOS performance against the modified Ryan_FC team across both short-run and aggregate match sets.

These reports are self-contained and can be viewed directly in the browser without running any scripts or setting up the project locally.
